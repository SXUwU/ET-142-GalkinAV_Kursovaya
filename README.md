# ET-142-GalkinAV_Kursovaya
Студент: Галкин Александр Васильевич
Группа: ЕТ-142

Курсовой проект на тему "Поиск и первичный анализ наборов данных по различным областям применения"
Содержит пять датасетов для разных задач:
1. Аудиодатасет
   Задача: Транскрибация и протоколирование контрольного мероприятия инспектором
   Ссылка на датасет: https://github.com/salute-developers/golos/blob/master/golos/README_ru.md
2. Изображения
   Задача: Распознавание граффити на памятниках
   Ссылка на датасет: https://zenodo.org/records/3238357
3. Табличные данные
   Задача: Определение типа банковской транзакции
   Исходные данные: Данные о транзакциях: сумма, время суток, день недели, тип счёта (дебетовый/кредитный), место (город/онлайн), предыдущие транзакции.
   Требуется: Построить модель классификации, определяющую тип транзакции: оплата, перевод, снятие.
   Ссылка на датасет: [https://www.kaggle.com/datasets/marusagar/bank-transaction-fraud-detection](https://github.com/rskworld/fraud-detection)
4. Текст
   Задача: Классификация спама по тексту электронного письма.
   Исходные данные: Тексты электронных писем, преобразованные в вектор признаков на основе словаря из 5 000 слов. Разметка: спам / не спам.
   Требуется: Построить модель классификации, определяющую, является ли письмо спамом.
   Ссылка на датасет: https://www.kaggle.com/datasets/purusinghvi/email-spam-classification-dataset?resource=download
5. Временные ряды
   Задача: Раннее обнаружение перегрева подшипника по тренду температуры.
   Исходные данные: Многомерные температурные временные ряды с датчиков.
   Требуется: Разработать модель прогноза аномалии на основе трендов и градиентов температуры.
   Цель: обнаружение неисправности за 10–30 минут
   Ссылка на датасет: https://data.mendeley.com/datasets/5hcdd3tdvb/6
